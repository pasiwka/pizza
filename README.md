# 🍕 React Pizza

**React Pizza** — это приложение для онлайн-заказа пиццы, построенное на **React 19**, **TypeScript** и **Redux Toolkit**. Приложение предоставляет удобный каталог с фильтрацией, сортировкой, поиском, пагинацией, корзиной и детальной страницей каждой пиццы.

---

## Функционал

### Каталог пицц

- Главная страница со списком пицц, загружаемых с сервера (MockAPI)
- **Скелетоны (Skeleton)** — анимация загрузки с помощью `react-content-loader`
- Обработка ошибок загрузки с выводом сообщения пользователю

### Фильтрация и сортировка

- **Категории**: Все, Мясные, Вегетарианская, Гриль, Острые, Закрытые
- **Сортировка**: по популярности (↑/↓), цене (↑/↓), алфавиту (↑/↓)
- **Поиск**: поиск пицц по названию с дебаунсом (`lodash.debounce`)
- **Пагинация**: постраничная навигация (4 пиццы на страницу) через `react-paginate`

### Детальная страница пиццы

- Маршрут `/pizzas/:id` — загрузка данных о конкретной пицце
- Отображение фото, названия, размеров, описания и цены
- Обработка ошибки при несуществующем ID с редиректом на главную

### Корзина

- Добавление пиццы в корзину (увеличение количества при повторном добавлении)
- Уменьшение количества / удаление позиции
- Очистка всей корзины с подтверждением
- Автоматический пересчёт общей суммы и количества
- **Сохранение корзины в `localStorage`** — данные не теряются при перезагрузке
- Пустая корзина с призывом вернуться на главную

### Синхронизация с URL

- Параметры фильтрации, сортировки и номер страницы сохраняются в query-строке
- Можно поделиться ссылкой на конкретный набор фильтров
- Парсинг URL при загрузке для восстановления состояния (`qs`)

### Lazy Loading

- Страницы корзины, детальной информации и 404 загружаются лениво (`React.lazy` + `Suspense`)
- Разделение кода на чанки (code splitting)

## Архитектура Redux

### Store (`redux/store.ts`)

- **filter** — состояние фильтров (категория, сортировка, поиск, страница)
- **cart** — состояние корзины (товары, общая сумма)
- **pizza** — состояние каталога (список пицц, статус загрузки, ошибка)

### Slice пицц (`redux/pizzas/slice.ts`)

- Асинхронный thunk `fetchPizzas` для загрузки с сервера
- Три статуса: `loading`, `success`, `error`
- Экспорт селектора `selectPizza`

### Slice фильтров (`redux/filter/slice.ts`)

- `setCategoryId` — выбор категории
- `setSortId` — выбор сортировки
- `setSearchValue` — поисковый запрос
- `setCurrentPage` — номер страницы
- `setFilters` — восстановление всех фильтров из URL

### Slice корзины (`redux/cart/slice.ts`)

- `addItems` — добавление / увеличение количества
- `minusItem` — уменьшение количества (автоудаление при 0)
- `removeItems` — удаление позиции
- `clearItems` — очистка корзины
- Автоматическое сохранение в `localStorage` через Header

## API

Приложение использует **[MockAPI.io](https://mockapi.io/)** в качестве бэкенда.

**Эндпоинты:**

- `GET /items` — список пицц (с параметрами `page`, `limit`, `category`, `sortBy`, `order`, `search`)
- `GET /items/:id` — детальная информация о пицце
