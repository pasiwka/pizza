# Пиццерия 🍕

**Pizza** — это React-приложение для онлайн-заказа пиццы с удобным каталогом, фильтрацией, поиском и корзиной.

## Функционал

- **Каталог пицц** — главная страница со списком всех доступных пицц, загружаемых с сервера
- **Фильтрация по категориям** — выбор категории (мясные, вегетарианские, гриль и т.д.)
- **Сортировка** — сортировка по популярности, цене (по возрастанию/убыванию) и алфавиту
- **Поиск** — поиск пицц по названию
- **Пагинация** — постраничная навигация по каталогу
- **Карточка пиццы** — детальная страница каждой пиццы с фото, описанием, размерами и ценой
- **Корзина** — добавление/удаление пицц, изменение количества, очистка корзины
- **Скелетоны (Skeleton)** — анимация загрузки при ожидании данных с сервера
- **Синхронизация с URL** — параметры фильтрации, сортировки и номер страницы сохраняются в URL (можно поделиться ссылкой)

## Технологии

| Технология               | Назначение                       |
| ------------------------ | -------------------------------- |
| **React 19**             | UI-библиотека                    |
| **React Router v6**      | Клиентская маршрутизация         |
| **Redux Toolkit**        | Управление состоянием приложения |
| **React Redux**          | Связка React с Redux             |
| **Axios**                | HTTP-запросы к API               |
| **SCSS**                 | Стилизация компонентов           |
| **React Paginate**       | Пагинация                        |
| **React Content Loader** | Skeleton-анимация                |
| **qs**                   | Парсинг и генерация query-строк  |
| **lodash.debounce**      | Дебаунс поля поиска              |

## API

Приложение использует [MockAPI.io](https://mockapi.io/) в качестве бэкенда для получения данных о пиццах.

## Роутинг

| Маршрут       | Страница                 |
| ------------- | ------------------------ |
| `/`           | Главная (каталог пицц)   |
| `/cart`       | Корзина                  |
| `/pizzas/:id` | Детальная страница пиццы |
| `*`           | Страница 404             |
